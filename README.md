# Pretext + MUI Next.js Demo

This project demonstrates a Next.js app using Material-UI (MUI) and the @chenglou/pretext library for advanced text flow. Users can upload a transparent PNG image of a person, and the text flows around the image in two columns with a serif font.

## Features

- Next.js (TypeScript, App Router)
- Material-UI (MUI) for modern, accessible UI
- Image upload (PNG, transparent background recommended)
- Two-column, serif-font text layout
- Responsive and clean design

## Getting Started

1. Install dependencies (if not already):

   ```bash
   npm install
   ```

2. Run the development server:

   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

- Replace the sample text or default image in `src/app/page.tsx` as needed.
- Style overrides can be made using MUI's `sx` prop or by editing the styled components.
